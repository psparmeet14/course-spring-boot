**Steps to create Java Apps**

1. Model Class (C)
2. IDAO (I) - Data Access Layer
3. DAOImpl (C)
4. IService (I) - Service Layer
5. ServiceImpl (C)
6. Controller - Presentation Layer
7. Utility Class (C)
8. Validator Class (C)
9. Service Provider (C) or Service Consumer (C) - Integration Layer
10. View (C)
